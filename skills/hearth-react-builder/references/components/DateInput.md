# DateInput

Day/month/year split input for entering known dates (e.g. date of birth, start date). For calendar-based date browsing, use [`DatePicker`](DatePicker.md) instead.

```tsx
import { DateInput } from '@utilitywarehouse/hearth-react';

<DateInput label="Date of birth" />
```

Same label/validation props as [`TextInput`](TextInput.md).
