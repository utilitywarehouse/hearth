## Custom label

If you need to have a custom label you can pass a component to the `label`
prop, render it separately, or use the underlying `Fieldset` & `FieldsetLegend`
components directly.

The necessary aria props, such as `aria-labelledby`, `aria-describedby` and
`aria-errormessage` are handled internally, so if you do choose to render a
separate label, or even not to have a visual label, either `aria-label` or
`aria-labelledby` should be provided.
