# Common props

These are props that can have common usage across multiple components,
additional to the components own set of specialized props.

For instance `padding`, `margin` & `color` are all shared across many
components, therefore they are defined separately and included alongside
specific component props.

It will be pointed out in component documentation when these common props are
used.
