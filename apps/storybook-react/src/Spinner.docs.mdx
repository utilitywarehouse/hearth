## Size

The size prop is responsive, so you can render different sizes at different breakpoints.

```tsx
<Spinner size={{ mobile: 'sm', tablet: 'md', desktop: 'lg' }} />
```

## API

| Prop   | Type                                       |
| ------ | ------------------------------------------ |
| `size` | `Responsive<"xs" \| "sm" \| "md" \| "lg">` |
