# Colour props

## Color

The `color` prop accepts the following semantic colour values: `primary`,
`secondary`, `brand`, `affirmative`, `inverted`.

```tsx
<Box color="primary" />
<Box color="secondary" />
<Box color="brand" />
<Box color="affirmative" />
<Box color="inverted" />
```

You can also pass any of the available Hearth prefixed CSS custom properties
from the Hearth Tokens library.
You cannot pass raw strings to the `color` prop.

```tsx
import { Box } from '@utilitywarehouse/hearth-react'
import { semantic } from '@utilitywarehouse/hearth-tokens/browser';

// token value
<Box color="primary" />

// Hearth prefixed CSS custom property
<Box color="var(--h-color-cashback-lilac-500)" />

// Hearth browser tokens
<Box color={semantic.feedback.positive.foreground.subtle}/>

// This will show an error as the string pixel value is not valid
// <Box color='rebeccapurple'/>
```

| Prop    | Type                                                                                         |
| ------- | -------------------------------------------------------------------------------------------- |
| `color` | `'primary' \| 'secondary' \| 'brand' \| 'affirmative' \| 'inverted' \| 'var(--h-${string})'` |

## Border colour

The border colour props accept the following semantic colour values: `strong`, `subtle`.

```tsx
<Box borderColor="strong" />
<Box borderColor="subtle" />
```

This applies to all of the available border colour values:

- `borderColor`
- `borderTopColor`
- `borderRightColor`
- `borderBottomColor`
- `borderLeftColor`

You can also pass any of the available Hearth prefixed CSS custom properties from the Hearth Tokens library.
You cannot pass raw strings to the border colour props.

```tsx
import { Box } from '@utilitywarehouse/hearth-react'
import { semantic } from '@utilitywarehouse/hearth-tokens/browser';

// token value
<Box borderColor="subtle" />

// Hearth prefixed CSS custom property
<Box borderColor="var(--h-color-grey-500)" />

// Hearth browser tokens
<Box
  borderColor={semantic.feedback.positive.border}
  borderWidth="1"
  borderStyle="solid"
  borderRadius="lg"
/>

// This will show an error as the string pixel value is not valid
// <Box borderColor='rebeccapurple'/>
```

| Prop                | Type                                           |
| ------------------- | ---------------------------------------------- |
| `borderColor`       | `'strong' \| 'subtle' \| 'var(--h-${string})'` |
| `borderTopColor`    | `'strong' \| 'subtle' \| 'var(--h-${string})'` |
| `borderRightColor`  | `'strong' \| 'subtle' \| 'var(--h-${string})'` |
| `borderBottomColor` | `'strong' \| 'subtle' \| 'var(--h-${string})'` |
| `borderLeftColor`   | `'strong' \| 'subtle' \| 'var(--h-${string})'` |
