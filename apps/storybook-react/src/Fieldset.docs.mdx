## Grouping inputs

The `Fieldset` component should be used to group related form inputs.

```tsx
<Fieldset label="Personal details">
  <TextInput label="First name" required />
  <TextInput label="Middle name(s)" />
  <TextInput label="Last name" required />
</Fieldset>
```

## Custom label

You can render a custom label by passing a React node to the label prop.

```tsx
<Fieldset label={<Heading as="h2">Personal details</Heading>}>
  <TextInput label="First name" required />
  <TextInput label="Middle name(s)" />
  <TextInput label="Last name" required />
</Fieldset>
```

If your label for the group has to be outside the fieldset then you should use
an aria label to associate it with the fieldset group.

```tsx
<Flex direction="column" gap="300">
  <Heading as="h2" id="personal-details">
    Personal details
  </Heading>
  <Fieldset {...args} aria-labelledby="personal-details">
    <TextInput label="First name" required />
    <TextInput label="Middle name(s)" />
    <TextInput label="Last name" required />
  </Fieldset>
</Flex>
```

## Accessibility

If the `Fieldset` API doesn't meet your needs, please use it with the `Legend`
component where possible, ensuring the `Legend` is the first child of the
`Fieldset`, otherwise the association between the two elements will be broken.

When doing this you should not use the supplementary text props, such as `helperText`.

```tsx
<Fieldset>
  <Legend>
    <Heading as="h2">Personal details</Heading>
  </Legend>
  <TextInput label="First name" required />
  <TextInput label="Middle name(s)" />
  <TextInput label="Last name" required />
</Fieldset>
```

## API

This component is based on the `fieldset` element and supports the following common props:

- Margin

| Prop               | Type                   |
| ------------------ | ---------------------- |
| `label`            | `string`               |
| `helperText`       | `string`               |
| `validationText`   | `string`               |
| `validationStatus` | `"valid" \| "invalid"` |
