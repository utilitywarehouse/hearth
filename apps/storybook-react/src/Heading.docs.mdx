## Sizes

The size prop is not responsive, as the sizes themselves are already responsive
where necessary.

## API

This component is based on the `h2` element.

| Prop            | Type                                                               |
| --------------- | ------------------------------------------------------------------ |
| `as`            | `"h1" \| "h2" \| "h3" \| "h4"`                                     |
| `asChild`       | `boolean`                                                          |
| `size`          | `'sm' \| 'md' \| 'lg' \| 'xl'`                                     |
| `align`         | `Responsive<"left" \| "center" \| "right">`                        |
| `textTransform` | `Responsive<"none" \| "capitalize" \| "uppercase" \| "lowercase">` |
| `wrap`          | `Responsive<"wrap" \| "nowrap" \| "balance" \| "pretty">`          |
