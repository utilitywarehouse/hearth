## Sizes

The size prop is responsive, so you can render different sizes at different breakpoints.

```tsx
<DetailText size={{ mobile: 'sm', tablet: 'md', desktop: 'lg'}}>{...}</DetailText>
```

## API

This component is based on the `span` element.

| Prop      | Type                                                      | Description                                                                                                                                    | Default  |
| --------- | --------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- | -------- |
| `as`      | `"p" \| "div" \| "span"`                                  | Shorthand for changing the default rendered element into a semantically appropriate alternative. Cannot be used in combination with `asChild`. | `"span"` |
| `asChild` | `boolean`                                                 | Change the default rendered element for the one passed as a child, merging their props and behavior.                                           | `false`  |
| `size`    | `'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl' \| '3xl' \| '4xl'` | Set the text size styles.                                                                                                                      | `"md"`   |
| `align`   | `Responsive<"left" \| "center" \| "right">`               | Set the text-align.                                                                                                                            |          |
