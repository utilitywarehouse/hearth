## Padding

All the padding props use the [space design tokens](https://hearth.prod.uw.systems/?path=/docs/tokens_space--docs#available-space-tokens) only.

```tsx
<Box padding="100" />
<Box padding={{ mobile: "100", desktop: "300" }}/>
```
