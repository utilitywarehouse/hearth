## API

This component is based on the `label` element, and supports the following common props:

- Margin
- Text transform

| Prop                | Type                  |
| ------------------- | --------------------- |
| `as`                | `"label" \| "span"`   |
| `labelVariant`      | `"body" \| "heading"` |
| `disabled`          | `boolean`             |
| `disableUserSelect` | `boolean`             |
