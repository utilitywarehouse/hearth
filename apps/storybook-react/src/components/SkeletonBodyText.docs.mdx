# SkeletonBodyText

Use `SkeletonBodyText` for text-shaped loading placeholders where the final content will be body copy.

## Lines

Use the `lines` prop to render multiple skeleton rows.

```tsx
<SkeletonBodyText lines={3} />
```

Each rendered line uses the same `BodyText` typography sizing so the placeholder height tracks the final content more closely.

<Canvas of={Stories.SingleLine} />

## Size and Weight

`SkeletonBodyText` is based on `BodyText`, so you can use `size` and `weight` to match the expected final content.

```tsx
<SkeletonBodyText size="lg" weight="semibold" lines={2} />
```

## API

This component is based on `BodyText` and supports the same common props, including:

- Margin
- Color
- Text align
- Text wrap
- Text transform

| Prop     | Type                                            |
| -------- | ----------------------------------------------- |
| `size`   | `Responsive<"sm" \| "md" \| "lg">`              |
| `weight` | `Responsive<"regular" \| "semibold" \| "bold">` |
| `lines`  | `number`                                        |
