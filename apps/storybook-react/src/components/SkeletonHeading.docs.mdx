# SkeletonHeading

Use `SkeletonHeading` for heading-shaped loading placeholders where the final content will be a title or section heading.

## Size

`SkeletonHeading` is based on `Heading`, so the `size` prop should match the final heading style you expect to render.

```tsx
<SkeletonHeading size="lg" />
```

This keeps the placeholder height aligned with the eventual heading rhythm.

<Canvas of={Stories.ContentBlock} />

## API

This component is based on `Heading` and supports the same common props, including:

- Margin
- Text transform
- Text align
- Text wrap

| Prop      | Type                                    |
| --------- | --------------------------------------- |
| `as`      | `"h1" \| "h2" \| "h3" \| "h4"`          |
| `asChild` | `boolean`                               |
| `size`    | `"sm" \| "md" \| "lg" \| "xl" \| "2xl"` |
