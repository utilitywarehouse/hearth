## When to use

Use the Verification Input to allow users enters a one-time password (OTP) sent via SMS, email, or authenticator apps.

- Use when **verifying user identity** during login, sign-up, or sensitive actions.
- Use when adding an **extra layer of security** for multi-factor authentication (MFA).

## When not to use

- Do not use when allowing users to **create or set passwords**. Use `PasswordInput` instead.
