# ValidationText

`ValidationText` should be used with form field components to display validation messages.

<Canvas of={Stories.Playground} sourceState="hide" />

## API

This component is based on the `span` element and supports the following common props:

- <StorybookLink to="common-props-margin">Margin</StorybookLink>
- <StorybookLink to="common-props-text">Text transform</StorybookLink>
- <StorybookLink to="common-props-text">Text align</StorybookLink>

| Prop                | Type                   |
| ------------------- | ---------------------- |
| `status`            | `'valid' \| 'invalid'` |
| `disableUserSelect` | `boolean`              |
