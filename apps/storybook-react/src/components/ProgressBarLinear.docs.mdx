## Usage

Use the `ProgressBarLinear` component to display the progress of a task or process.
The component requires a `value` prop (number from 0-100) and accepts `ProgressBarLabel` and
`ProgressBarValue` as children to display contextual information.

```tsx
<ProgressBarLinear value={90}>
  <ProgressBarLabel label="Upload progress" />
  <ProgressBarValue value="90%" />
</ProgressBarLinear>
```

## Type

The `type` prop controls the visual appearance of the progress bar:

- `default` - Progress indicator (default)
- `success` - Progress indicator for completed or successful states
- `danger` - Progress indicator for critical or warning states

```tsx
<ProgressBarLinear type="success" value={100}>
  <ProgressBarLabel label="Complete" />
  <ProgressBarValue value="100%" />
</ProgressBarLinear>
```

## Value

The `value` prop accepts a number between 0 and 100, representing the percentage
of progress. Values outside this range will be automatically clamped.

```tsx
<ProgressBarLinear value={45}>
  <ProgressBarLabel label="Download progress" />
  <ProgressBarValue value="45%" />
</ProgressBarLinear>
```

## Label and Value

Use `ProgressBarLabel` to display descriptive text about what is progressing,
and `ProgressBarValue` to show the current value. Both are optional but
recommended for clarity.

```tsx
<ProgressBarLinear value={75}>
  <ProgressBarLabel label="Processing files" />
  <ProgressBarValue value="75 / 100" />
</ProgressBarLinear>
```

## Accessibility

The component includes proper ARIA attributes for screen readers:

- The main container has `role="group"` with `aria-label="Progress indicator"` which can be overriden
- The progress bar itself has `role="progressbar"` with `aria-valuenow`, `aria-valuemin`,
  `aria-valuemax`, and `aria-label` attributes

This ensures that assistive technologies can properly announce the progress state
and context to users.

## Animation

The progress bar fill animates smoothly when the value changes, using a 0.3s
ease-in-out transition. This provides visual feedback without being distracting.
