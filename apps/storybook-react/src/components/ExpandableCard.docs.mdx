# ExpandableCard

Use `ExpandableCard` to show a summary header with an optional leading icon and helper text, which
can be expanded to reveal additional content. It is built on the
[Radix UI Collapsible](https://www.radix-ui.com/primitives/docs/components/collapsible) primitive.

## Leading content

Use the `leadingContent` prop to render an icon or other element to the left of the heading. When
using an icon, pass `aria-hidden` to ensure it is not announced to screen readers.

<Canvas of={Stories.WithLeadingContent} />

## Helper text

The `helperText` prop renders a secondary line of text below the heading.

<Canvas of={Stories.WithoutHelperText} />

## Default open

Use `defaultOpen` to render the card in the open state on initial mount. This is an uncontrolled
approach — the card manages its own open/closed state internally after mount.

```tsx
<ExpandableCard heading="Heading" helperText="Helper text" defaultOpen>
  {children}
</ExpandableCard>
```

<Canvas of={Stories.DefaultOpen} />

## Controlled

Use the `open` and `onOpenChange` props together to control the open state externally.

```tsx
const [open, setOpen] = React.useState(false);

<ExpandableCard
  heading="Heading"
  helperText="Helper text"
  open={open}
  onOpenChange={setOpen}
>
  {children}
</ExpandableCard>
```

## Disabled

The `disabled` prop prevents the card from being opened or closed.

## Accessibility

Follows the [Disclosure (Show/Hide) WAI-ARIA design pattern](https://www.w3.org/WAI/ARIA/apg/patterns/disclosure/).

  ))}

## API

This component is based on the `div` element and supports the following common props:

- <StorybookLink to="common-props-margin">Margin</StorybookLink>

| Name             | Type                          |
| ---------------- | ----------------------------- |
| `heading`        | `string` (required)           |
| `helperText`     | `string`                      |
| `leadingContent` | `ReactNode`                   |
| `open`           | `boolean`                     |
| `defaultOpen`    | `boolean`                     |
| `onOpenChange`   | `(open: boolean) => void`     |
| `disabled`       | `boolean`                     |
