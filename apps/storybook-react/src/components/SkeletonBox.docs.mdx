# SkeletonBox

Use `SkeletonBox` for non-text loading placeholders where the final layout is box-like, such as media, cards, tiles, and blocks.

## Usage

`SkeletonBox` is based on `Box`, so it can take the same layout-oriented props for size, spacing, border radius, and positioning.

```tsx
<SkeletonBox width="160px" height="96px" borderRadius="sm" />
```

Use `SkeletonBox` when you need explicit control over shape and dimensions.

<Canvas of={Stories.Blocks} />

## API

This component is based on `Box` and supports the same common props, including:

- Padding
- Margin
- Position
- Size
- Border radius
- Opacity

| Prop      | Type              |
| --------- | ----------------- |
| `as`      | `"div" \| "span"` |
| `asChild` | `boolean`         |
