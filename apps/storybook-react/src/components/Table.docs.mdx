## Usage

- `Table` groups the `TableHeader` and `TableBody` components.
- `TableContainer` groups the `Table` and `TablePagination` components, and provides UI variants.
- `TableHeader` for table column headers
- `TableBody` provides the body of the table.
- `TableRow` to wrap rows of data cells
- `TableHeaderCell` provides the header cell, which can be used within both `TableHeader` and `TableRow` components.
- `TableCell` for data cells within a `TableRow`.
- `TablePagination` provides a data cell for use within a `TableRow` component.

## Container

To include different visual variants you must wrap your `Table` in a `TableContainer` component.

`TableContainer` supports the following variants, based on the `Card` component:

- `undefined` - No `Card` container, minimal styling
- `subtle` - Subtle `Card` container
- `emphasis` - Emphasis `Card` container

```tsx
<TableContainer variant="subtle">
  <Table>
    <TableHeader>
      <TableHeaderCell>Name</TableHeaderCell>
      <TableHeaderCell>Occupation</TableHeaderCell>
    </TableHeader>
    <TableBody>
      <TableRow>
        <TableHeaderCell row>Toni Morrison</TableCell>
        <TableCell>Writer</TableCell>
      </TableRow>
    </TableBody>
  </Table>
</TableContainer>
```

## Pagination

To include pagination, you need to wrap the `Table` & `TablePagination` components in a `TableContainer`

```tsx

[...]

const [currentPage, setCurrentPage] = useState(1);
const itemsPerPage = 5;

const totalPages = Math.ceil(personalDetails.length / itemsPerPage);
const startIndex = (currentPage - 1) * itemsPerPage;
const endIndex = startIndex + itemsPerPage;
const currentData = personalDetails.slice(startIndex, endIndex);

[...]

<TableContainer variant="subtle">
  <Table>
    <TableHeader>{...}</TableHeader>
    <TableBody>{...}</TableBody>
  </Table>
  <TablePagination
    currentPage={currentPage}
    totalPages={totalPages}
    onPageChange={setCurrentPage}
  />
</TableContainer>
```

## Header cells

The `TableHeaderCell` component is used for column header cell within the `TableHeader` component.

When the `row` prop is set it can also be used as the first cell in a `TableRow`.

```tsx
[...]
<TableHeader>
  <TableHeaderCell>Name</TableHeaderCell>
  <TableHeaderCell>Occupation</TableHeaderCell>
</TableHeader>
<TableBody>
  <TableRow>
    <TableHeaderCell row>Toni Morrison</TableCell>
    <TableCell>Writer</TableCell>
  </TableRow>
</TableBody>
[...]
```

## API

### TableContainer API

This component is based on the `div` element and supports the following common props:

- Margin

| Prop      | Type                                    |
| --------- | --------------------------------------- |
| `variant` | `'undefined' \| 'subtle' \| 'emphasis'` |

### Table API

This component is based on the `table` element and supports the following common props:

- Margin

### TableHeader API

This component is based on the `thead` element.

### TableHeaderCell API

This component is based on the `th` element.

### TableBody API

This component is based on the `tbody` element.

### TableRow API

This component is based on the `tr` element.

### TableCell API

This component is based on the `td` element.

### TablePagination API

This component is based on the `Pagination` component.
