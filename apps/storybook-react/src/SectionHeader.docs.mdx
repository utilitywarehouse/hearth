## Trailing content

Use the `trailingContent` prop to include a `Link` or `Badge` component.

```tsx
import { Link, SectionHeader } from '@utilitywarehouse/hearth-react';
import { ChevronRightSmallIcon } from '@utilitywarehouse/hearth-react-icons';

[...]

<SectionHeader
  {...args}
  link={
    <Link>
      Link text
      <ChevronRightSmallIcon />
    </Link>
  }
/>
```

## API

| Prop              | Type                                    |
| ----------------- | --------------------------------------- |
| `heading`         | `string`                                |
| `headingElement`  | `"div" \| "h1" \| "h2" \| "h3" \| "h4"` |
| `helperText`      | `string`                                |
| `trailingContent` | `ReactNode`                             |
