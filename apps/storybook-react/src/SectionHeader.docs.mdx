## Link

You can include a `Link` component as a child of the `SectionHeader`.

```tsx
import { Link, SectionHeader } from '@utilitywarehouse/hearth-react';
import { ChevronRightSmallIcon } from '@utilitywarehouse/hearth-react-icons';

[...]

<SectionHeader heading="New Section" helperText="Further details">
  <Link>
    See more
    <ChevronRightSmallIcon />
  </Link>
</SectionHeader>;
```

## API

| Prop             | Type                                    |
| ---------------- | --------------------------------------- |
| `heading`        | `string`                                |
| `headingElement` | `"div" \| "h1" \| "h2" \| "h3" \| "h4"` |
| `helperText`     | `string`                                |
