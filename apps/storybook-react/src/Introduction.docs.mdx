## Getting started

Assuming you already have React installed and your project set up, to get started install the library and it's peer dependencies.

```console
yarn add @utilitywarehouse/hearth-react @utilitywarehouse/hearth-tokens @utilitywarehouse/hearth-fonts @utilitywarehouse/hearth-react-icons
```

We also advise you install the Hearth baseline CSS styles

```console
yarn add @utilitywarehouse/hearth-css-reset
```

You should then add these imports in the root of your application, along with the Hearth React styles.

```tsx
import '@utilitywarehouse/hearth-css-reset';
import '@utilitywarehouse/hearth-fonts';
import '@utilitywarehouse/hearth-tokens/index.css';
import '@utilitywarehouse/hearth-react/styles.css';
```
