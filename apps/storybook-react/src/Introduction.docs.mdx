## Getting started

You should already have your project set up, with React, either v18 or v19.

You can then install the necessary Hearth libraries:

- Hearth React
- Hearth CSS Reset
- Hearth Tokens
- Hearth React Icons
- Hearth Fonts

```console
yarn add @utilitywarehouse/hearth-react @utilitywarehouse/hearth-css-reset @utilitywarehouse/hearth-tokens @utilitywarehouse/hearth-fonts @utilitywarehouse/hearth-react-icons
```

or

```console
npm install @utilitywarehouse/hearth-react @utilitywarehouse/hearth-css-reset @utilitywarehouse/hearth-tokens @utilitywarehouse/hearth-fonts @utilitywarehouse/hearth-react-icons
```

or

```console
pnpm add @utilitywarehouse/hearth-react @utilitywarehouse/hearth-css-reset @utilitywarehouse/hearth-tokens @utilitywarehouse/hearth-fonts @utilitywarehouse/hearth-react-icons
```

You should then add the following imports in the root of your application:

```tsx
import '@utilitywarehouse/hearth-css-reset';
import '@utilitywarehouse/hearth-fonts';
import '@utilitywarehouse/hearth-tokens/index.css';
import '@utilitywarehouse/hearth-react/styles.css';
```
