# Hearth SVG Assets

Static assets library for illustrations, scenes, mascots, and logos used across UW products.<br/>
Current version: v{version}

Hearth SVG Assets is a collection of SVG files (not React components):

- Logos (full/condensed)
- Spots (small illustrations)
- Scenes (large illustrations)
- Mascots

- [Installation](#installation)
- [Usage](#usage)
- [Download from Storybook](#download-from-storybook)

## Installation

```console
pnpm add @utilitywarehouse/hearth-svg-assets
```

## Usage

When bundling SVGs via a bundler, you can import a file path and use it in an img tag:

```tsx
import SceneBroadband from '@utilitywarehouse/hearth-svg-assets/lib/scene-broadband-light.svg';

<img src={SceneBroadband} alt="Broadband scene" />;
```

If you need dark mode support, you can import both light and dark versions and switch between them based on your app's color mode:

```tsx
import SceneBroadbandLight from '@utilitywarehouse/hearth-svg-assets/lib/scene-broadband-light.svg';
import SceneBroadbandDark from '@utilitywarehouse/hearth-svg-assets/lib/scene-broadband-dark.svg';

const isDarkMode = getDarkModeSomehow(); // determine if dark mode is active

<img src={isDarkMode ? SceneBroadbandDark : SceneBroadbandLight} alt="Broadband scene" />;
```

If your setup doesn’t support importing .svg as a URL, copy the raw SVG from the asset story, or download it from the Search page below.

## Download from Storybook

Use the “Search Assets” page to locate an asset and click Download to save the SVG locally.
