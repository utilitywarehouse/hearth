# Hearth JSON Assets

Current version: v{version}

These are JSON animation assets for Lottie-style rendering in web and native mobile apps.

## Installation

```console
pnpm add @utilitywarehouse/hearth-json-assets
```

## Usage

Import JSON animation data directly and pass it to your animation renderer:

```tsx
import Lottie from 'lottie-react';
import SpotProcessCompleteFunctional from '@utilitywarehouse/hearth-json-assets/lib/animated-spot-process-complete-functional.json';

function Example() {
  return <Lottie animationData={SpotProcessCompleteFunctional} loop={false} />;
}
```

If you do not use `lottie-react`, you can pass the imported JSON object to any compatible animation player.

## Accessibility

When using these assets in an application, ensure that you do not set them to
loop indefinitely, as this can cause issues for users with cognitive
disabilities.
