## Source

The icons in these packages are all generated from designs in the [UW Icons Figma library](https://www.figma.com/file/WDFaQF9EMtS7MjuIpjDVpf/UW-Icons?node-id=58-161&t=4nIdPbiaJQZQUw4S-0).

The source code is available in the [Icons GitHub repo](https://github.com/utilitywarehouse/icons).

## Sizes

There are 3 icon sizes available. Not all icons are represented by each size.

- Small: 16px
- Medium: 24px
- Large: 48px

## Variants

There are 3 icon variants available. Not all icons are represented by each variant.

- Regular
- Filled
- Contained

## Naming conventions

The icons follow a naming convention.

- Name: the descriptive name of the icon will be the first part; ie. `Add`, `Cross`, `Tick`.
- Size: the size of the icon will be next; ie. `Small`, `Medium`, `Large`.
- Variant: if the icon is a variant that is not `Regular`, that variant name will be used next; ie. `Filled`, `Contained`.

## Packages

### SVG icons

UW iconography provided as a set of optimized svg assets, autogenerated from the
[Figma Icons library](https://www.figma.com/file/WDFaQF9EMtS7MjuIpjDVpf/UW-Icons?node-id=58-161&t=3g4P8SjA2RlEUxp9-0).

```
yarn add @utilitywarehouse/svg-icons
```

### React icons

UW iconography provided as a set of React components, autogenerated from the
`svg-icons` library.

```
yarn add @utilitywarehouse/react-icons
```

### React Native icons

UW iconography provided as a set of React Native components, autogenerated from the
`svg-icons` library.

```
yarn add @utilitywarehouse/react-native-icons
```

## Referencing icons from designs

As an engineer, when you are implementing a design you will need 3 pieces of
information to know which icon to use.

1. The icon name.
2. The icon size.
3. The icon variant.

The first 2 of these you will see in Figma as a name - size pair. For instance
**Tick - 16**. In Figma the size is numerical according to the key described
above, so you will need to translate 16 to `Small`, 24 to `Medium` and 48 to
`Large`.

In most cases it should be possible to know which icon variant you need by
looking at the icon in Figma and comparing it to the named icons in the
engineering docs. However if you need to check, you can click through to the
main component from Figma's dev mode, and check which variant container it is
in.
