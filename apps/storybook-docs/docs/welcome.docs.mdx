Hearth is the unified design system powering coherent, accessible user
interfaces across all our platforms. Built with scalability and flexibility in
mind, Hearth brings together our shared visual language, component libraries,
iconography, and design tokens — so teams can build faster with confidence and
consistency.

Whether you're working on web, mobile, or crafting brand assets, Hearth is your
central source of truth for all things design and UI implementation. And it's
all built with accessibility at its core.

## 🚀 Getting Started

To start using Hearth in your project:

1. **Pick your platform** – Choose the relevant library (React, React Native, etc.)
2. **Install the package** – Each library includes setup instructions and usage examples
3. **Follow the guidelines** – Use our accessible tokens, components, and styles as documented
4. **Stay up to date** – We ship updates regularly to improve consistency and expand functionality, follow us on slack [#help-design-systems](https://utilitywarehouse.slack.com/archives/C04MKBQ34PJ)

Whether you're a designer, developer, or both — Hearth is here to make your workflow smoother and your UI more delightful.
