# Border

Border radius and border width tokens used across Hearth components.

## Import

```js

import '@utilitywarehouse/hearth-tokens/css/border.css';

// Browser tokens for use in JavaScript or TypeScript web projects
import { border } from '@utilitywarehouse/hearth-tokens/browser';

// JS tokens for use in React Native projects
import { borderRadius, borderWidth } from '@utilitywarehouse/hearth-tokens/js';
```
