# Hearth Tokens

**v{version}**

Hearth Tokens is a platform-agnostic design tokens library providing an easy
way to access predefined values for spacing, colour, typography, shadows, borders,
radii, and more.

It is generated from the Figma source, and used by the UX Design team, as well
as in both the **Hearth React** and **Hearth React Native** component
libraries. It can also be consumed directly in your product applications,
whether web or native.

## Formats

Hearth Tokens are available in multiple formats to suit different platforms and use cases.

- **CSS** - CSS variables for use in web projects using CSS files
- **Browser** - JavaScript tokens referencing CSS variables, for use in web projects using CSS-in-JS
- **JS** - JavaScript tokens referencing raw values for use in React Native or any other JS/TS projects.

These can each be imported from the `hearth-tokens` package via the relevant subpath; `/css`, `/browser` or `/js`.

### CSS

- Provides CSS files that define the actual CSS custom properties (e.g. `--h-color-blue-500: #41a0ff;`, `--h-space-100: 8px;`).
- Use these when you want to use the tokens directly within your standard CSS, SCSS, or CSS Modules files.
- Use the `css` subpath to import: `@utilitywarehouse/hearth-tokens/css` or `@utilitywarehouse/hearth-tokens/css/{token-type}.css` (e.g., `@utilitywarehouse/hearth-tokens/css/color.css`, `@utilitywarehouse/hearth-tokens/css/space.css`)

### Browser

- Provides JavaScript objects which reference CSS variables (e.g. `'var(--h-color-blue-500)'`, `'var(--h-space-100)'`).
- Use these when you are working in a browser environment and want to apply styles using JavaScript (e.g. inline styles, CSS-in-JS libraries).
- Use the `browser` subpath to import: `@utilitywarehouse/hearth-tokens/browser` or `@utilitywarehouse/hearth-tokens/browser/{tokenType}` (e.g., `@utilitywarehouse/hearth-tokens/browser/color`, `@utilitywarehouse/hearth-tokens/browser/space`)
- **IMPORTANT** These require the CSS custom properties to exist in your project, either via the **Hearth React** library, or by importing the CSS definitions from the `hearth-tokens` package directly.

### JS

- Provides JavaScript objects which reference raw token values (e.g. hex codes for colours, pixel numbers for spacing).
- Use this when you need the actual value in any JavaScript/TypeScript environment (Node.js, React Native, Browser JS for calculations, etc.).
- Use the `js` subpath to import: `@utilitywarehouse/hearth-tokens/js` or `@utilitywarehouse/hearth-tokens/js/{tokenType}` (e.g. `@utilitywarehouse/hearth-tokens/js/color`, `@utilitywarehouse/hearth-tokens/js/space`)
