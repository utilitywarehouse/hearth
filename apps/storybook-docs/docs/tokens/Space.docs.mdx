# Space

Spacing values for use in layout and component spacing, such as padding,
margins, and gaps. These tokens should be used to define the _space_ around,
between or within elements, not the _size_ of elements.

## Import

```js
import { space } from '@utilitywarehouse/hearth-tokens/browser';
// or
import { space } from '@utilitywarehouse/hearth-tokens/js';
```

|                JS                |                Browser                |                CSS                 | Value |
| :------------------------------: | :-----------------------------------: | :--------------------------------: | :---: |
| space[{Object.keys(jsSpace)[0]}] | space[{Object.keys(browserSpace)[0]}] | { Object.values(browserSpace)[0] } |  4px  |
