# Component

Component tokens define the visual properties of individual components — spacing,
sizing, border radii, colours, and more. They reference semantic and primitive
tokens and should not be used directly in application code.

In browser environments all component tokens are available under the `components`
export as CSS custom properties. In JS/React Native, values are split into
`components.light` and `components.dark` to support dark mode.

## Import

```js
// CSS custom properties for use in CSS web projects
import '@utilitywarehouse/hearth-tokens/css/components.css';

// Browser tokens for use in JavaScript or TypeScript web projects
import { components } from '@utilitywarehouse/hearth-tokens/browser';

// JS tokens for use in React Native projects
import { components } from '@utilitywarehouse/hearth-tokens/js';
// components.light.accordion.gap
// components.dark.accordion.gap
```
