## Source

The icons in these packages are all generated from the [Hearth Icons Figma library](https://www.figma.com/design/x1DivEZ23UPZP7WXufHPjG/Hearth-Icons?node-id=4-138&t=GYzfYbXberQKesCr-0).

The source code is available in the [Hearth GitHub repo](https://github.com/utilitywarehouse/hearth).

## Sizes

There are 3 icon sizes available. Not all icons are represented by each size.

- Small: 20px
- Medium: 24px
- Large: 48px

## Naming conventions

The icons follow a naming convention.

- Name: the descriptive name of the icon will be the first part; ie. `Add`, `Cross`, `Tick`.
- Size: the size of the icon will be next; ie. `Small`, `Medium`, `Large`.

## Packages

### SVG icons

A set of optimized svg assets, autogenerated from the
[Hearth Icons Figma library](https://www.figma.com/design/x1DivEZ23UPZP7WXufHPjG/Hearth-Icons?node-id=4-138&t=GYzfYbXberQKesCr-0).

These icons are intended for use in non-React projects.

```
yarn add @utilitywarehouse/hearth-svg-icons
```

### React icons

A set of React components, autogenerated from the `hearth-svg-icons` library.

```
yarn add @utilitywarehouse/hearth-react-icons
```

### React Native icons

A set of React Native components, autogenerated from the `hearth-svg-icons` library.

```
yarn add @utilitywarehouse/hearth-react-native-icons
```
