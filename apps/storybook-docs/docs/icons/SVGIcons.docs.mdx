# Hearth SVG Icons

Current version: v{version}

Hearth Icons as optimized svg assets, autogenerated from the [UW Icons Figma library](https://www.figma.com/design/x1DivEZ23UPZP7WXufHPjG/Hearth-Icons?node-id=4-138&t=mDtT36KbCBo0LX9P-0).

While you can directly import this package into your application, it is not necessarily
intended to be used in this way, and exists more as a base to build framework
specific icon packages.

## Installation

```console
yarn add @utilitywarehouse/hearth-svg-icons
```

## Icon sizes

There are 3 sizes of icons:

- Small: 16px
- Medium: 24px
- Large: 48px

## Icon naming

The icon names are structured as:

> `[name]-[size]-icon.svg`

- `name` - Name of the icon from [Figma]('https://www.figma.com/file/WDFaQF9EMtS7MjuIpjDVpf/UW-Icons?node-id=27-230&t=PKen5tLHUHUzchVS-0') that is all lowercased and hyphen separated.
- `size` - Size of the icon that is one of small (20)/medium (24)/large (48). Note that some icons do not have all sizes available yet. Our designers are working to add missing ones to complete the collection.

## Usage

This library offers icons in SVG format. If you are using icons in a React
application, we recommend using the `@utilitywarehouse/hearth-react-icons` library,
which renders these same SVG icons.

If you want, you can import the SVG icons into your application, and use an `img` element:

```tsx
import AddMediumIcon from '@utilitywarehouse/hearth-svg-icons/lib/add-medium-icon.svg'

[...]

<img src={AddMediumIcon} alt="add">
```

However, you won't be able to control the SVG attributes by CSS, such as the icon color.
You're probably better off copying the raw SVG code from the Icons story.

See this [Using SVG](https://css-tricks.com/using-svg/) for more details, and options.
