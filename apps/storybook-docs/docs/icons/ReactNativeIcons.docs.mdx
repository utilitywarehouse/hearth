# Hearth React Native Icons

Current version: v{version}

Hearth Icons as optimized React Native components, generated from the `hearth-svg-icons` package.

## Installation

```console
yarn add @utilitywarehouse/hearth-react-native-icons
```

## Props

Each icon component supports the full range of [SVG attributes](https://developer.mozilla.org/en-US/docs/Web/SVG/Element/svg#attributes) as well as the following props.

<ArgTypes of={Stories} />

## Icon sizes

There are 3 sizes of icons:

- Small: 20px
- Medium: 24px
- Large: 48px

## Icon colours

The icon components don't have any predefined colours. The rendered SVG fill
colour is `currentColor` by default. You can change the icon colour using the
`color` prop, passing in any valid colour string. We recommend using the colour
values from the `hearth-design-tokens` package.

## Icon naming

The icon names are structured as:

> `[Name][Size]Icon.svg`

- `name` - Name of the icon from [Figma]('https://www.figma.com/file/WDFaQF9EMtS7MjuIpjDVpf/UW-Icons?node-id=27-230&t=PKen5tLHUHUzchVS-0') that is all lowercased and hyphen separated.
- `size` - Size of the icon that is one of small (16)/medium (24)/large (48). Note that some icons do not have all sizes available yet. Our designers are working to add missing ones to complete the collection.

## Usage

```tsx
import { AddMediumIcon } from '@utilitywarehouse/hearth-react-native-icons';

[...]

<AddMediumIcon />
```
