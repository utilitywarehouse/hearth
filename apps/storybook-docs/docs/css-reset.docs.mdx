# Hearth CSS Reset

A bare-bones CSS reset for UW web development.

Current version: v{version}

[View source](https://github.com/utilitywarehouse/hearth/tree/main/packages)

These styles are derived from the following CSS resets:

- [joshwcomeau.com](https://www.joshwcomeau.com/css/custom-css-reset/)
- [andy-bell.co.uk](https://andy-bell.co.uk/a-modern-css-reset/)

## Installation

```
yarn add @utilitywarehouse/hearth-css-reset
```

## Usage

```js
import '@utilitywarehouse/hearth-css-reset';
```
