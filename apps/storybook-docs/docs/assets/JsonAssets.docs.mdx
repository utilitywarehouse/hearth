# Hearth JSON Assets

Current version: v{version}

These are JSON animation assets for Lottie-style rendering in web and native mobile apps.

## Installation

```console
pnpm add @utilitywarehouse/hearth-json-assets
```

## Usage

Import JSON animation data directly and pass it to your animation renderer:

```tsx
import Lottie from 'lottie-react';
import SpotProcessCompleteFunctional from '@utilitywarehouse/hearth-json-assets/lib/animated-spot-process-complete-functional-light.json';

function Example() {
  return <Lottie animationData={SpotProcessCompleteFunctional} loop={false} />;
}
```

There are variations for both light and dark mode, indicated by the suffixes `-light` and `-dark` in the file names.

```tsx
import Lottie from 'lottie-react';
import SpotProcessCompleteDark from '@utilitywarehouse/hearth-json-assets/lib/animated-spot-process-complete-dark.json';
import SpotProcessCompleteLight from '@utilitywarehouse/hearth-json-assets/lib/animated-spot-process-complete-light.json';

function Example() {
  const isDarkMode = useIsDarkMode(); // Replace with your dark mode detection logic

  return (
    <Lottie
      animationData={isDarkMode ? SpotProcessCompleteDark : SpotProcessCompleteLight}
      loop={false}
    />
  );
}
```

If you do not use `lottie-react`, you can pass the imported JSON object to any compatible animation player.

## Accessibility

When using these assets in an application, ensure that you do not set them to
loop indefinitely, as this can cause issues for users with cognitive
disabilities.
