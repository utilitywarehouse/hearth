# Hearth Assets

Hearth Assets is a collection of static and animated assets including
illustrations and logos that can be used across platforms. It currently
includes two packages:

- Hearth SVG Assets: A collection of SVG icons and illustrations for use in web and
  native applications.
- Hearth JSON Assets: A collection of Lottie JSON animations for use in web and native
  applications.

## Photography

Use the Brandfolder CDN to access our image assets, including JPG and PNG
formats. [Read our guide on the wiki for more detailed info.](https://wiki.uw.systems/posts/using-brandfolder-cdn-images-in-our-apps-yhidpavm)
