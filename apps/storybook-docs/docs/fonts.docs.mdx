Font library for UI development at Utility Warehouse.<br/>

application.

## Getting Started

Add the hearth-fonts package to your project dependencies.

```console
yarn add @utilitywarehouse/hearth-fonts
# or
pnpm add @utilitywarehouse/hearth-fonts
# or
npm install --save @utilitywarehouse/hearth-fonts
```

You can then import the CSS within your app entry file or site component. This
will include all font families, weights & styles.

```js
import '@utilitywarehouse/hearth-fonts';
```

Alternatively you can select specific fonts, weights and styles, allowing you to
cut down on payload sizes to the last byte!

```js
// Include only DM Sans
import '@utilitywarehouse/hearth-fonts/dm-sans.css';
```

Finally, you can reference the font name in a CSS stylesheet, CSS Module, or CSS-in-JS.

```css
body {
  font-family: 'DM Sans', sans-serif;
}
```
