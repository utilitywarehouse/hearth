# Hearth Fonts

v{version}

Font library for UI development at Utility Warehouse.<br/>

All the font and CSS files you need to self-host the Utility Warehouse font families in your
application.

Get the performance gains of self-hosting the font, with woff2 & woff formats,
and the added reliability of versioning, without the hassle of managing font
assets.

_Add the fonts as a dependency, import into your project, and you're ready to go!_ 🚀

## Included Fonts

This package includes the following font families, styles and weights.

| **Font Family** | **Font Weights** | **Font Styles** |
| --------------- | ---------------- | --------------- |
| Comic Hams      | 600, 700, 900    | Normal          |
| DM Sans         | 400, 600, 700    | Normal, Italic  |
| DM Mono         | 500              | Normal, Italic  |
