Hearth Tokens is a platform-agnostic design tokens library covering spacing, colors, typography, shadows, radii, and
more. Delivered in multiple formats for seamless integration.

## Formats

Hearth Tokens are available in multiple formats to suit different platforms and use cases.

- **Browser JS** - JS tokens which reference the CSS variables.
- **JS** - JS tokens with raw values for use in React, React Native or any other JS/TS projects.
- **CSS** - CSS variables for use in web projects.

If your platform is not listed or you need a different format, please reach out to the Hearth team for support on the [#help-design-systems](https://utilitywarehouse.slack.com/archives/C04MKBQ34PJ) Slack channel.

## Installation

```bash
yarn add @utilitywarehouse/hearth-tokens
# or
pnpm add @utilitywarehouse/hearth-tokens
# or
npm install --save @utilitywarehouse/hearth-tokens
```
