## Usage Examples

Below are general examples. Refer to the specific documentation for each token type (Colours, Space, etc.) for detailed structures and available tokens.

### Raw JavaScript Values (`/js`)

```javascript
// Example: Using space tokens in React Native
import { StyleSheet } from 'react-native';
import { space } from '@utilitywarehouse/hearth-tokens/js';

const styles = StyleSheet.create({
  container: {
    padding: space[100], // Gets the raw pixel value, e.g., 8
    margin: space[200], // e.g., 16
  },
});

// Example: Using color tokens in a Node.js script
import { color } from '@utilitywarehouse/hearth-tokens/js';

function logPrimaryColor() {
  console.log('Primary color (light):', color.light.blue[500]); // Logs the hex code, e.g., '#41a0ff'
}
```

### CSS Definitions (`/css`)

**Step 1: Import CSS Definitions (do this once in your app)**

```javascript
// Import all CSS definitions
import '@utilitywarehouse/hearth-tokens/css';
```

**Step 2: Use CSS Variables in your CSS/SCSS/CSS Modules**

```css
/* styles.css */
.alert {
  padding: var(--h-space-200);
  background-color: var(--h-color-red-100);
  color: var(--h-color-red-900);
  border: 1px solid var(--h-color-red-300);
  border-radius: var(--h-space-50);
}

.page-title {
  color: var(--h-color-purple-700);
  margin-bottom: var(--h-space-400);
}
```

```css
/* component.module.css */
.container {
  padding: var(--h-space-300);
}

.text {
  color: var(--h-color-grey-700);
}
```

### Browser JS - CSS Variable Strings (`/browser`)

**Step 1: Import CSS Definitions (do this once in your app)**

```javascript
// Import all CSS definitions
import '@utilitywarehouse/hearth-tokens/css';
```

**Step 2: Use Browser JS Tokens**

```javascript
// Example: Using tokens in a React component with inline styles
import React from 'react';
import { color, space } from '@utilitywarehouse/hearth-tokens/browser';

const MyButton = ({ children }) => (
  <button
    style={{
      backgroundColor: color.blue[500], // 'var(--h-color-blue-500)'
      color: color.white, // 'var(--h-color-common-white)'
      padding: `${space[100]} ${space[200]}`, // 'var(--h-space-100) var(--h-space-200)'
      border: 'none',
      borderRadius: space[50], // 'var(--h-space-50)'
    }}
  >
    {children}
  </button>
);

// Example: Using tokens with styled-components
import styled from 'styled-components';
import { color, space } from '@utilitywarehouse/hearth-tokens/browser';

const StyledCard = styled.div`
  background-color: ${color.warmWhite[50]};
  padding: ${space[300]};
  border: 1px solid ${color.grey[200]};
  box-shadow: 0 ${space[25]} ${space[50]} ${color.grey[300]};
`;
```

Choose the format that best suits your project's architecture and the specific
task at hand. Remember to consult the individual token documentation pages for
the complete list of available tokens and their specific values or variable
names.
