## Getting Started

To use the font tokens in your project, you can import them from the `@utilitywarehouse/hearth-tokens` package. The tokens are available in multiple formats depending on your needs:

1.  **Raw Values (JS):** Provides raw font family strings (e.g., `'DM Sans'`), size numbers (e.g., `16`), and weight numbers (e.g., `400`). Use this in any JavaScript environment (Node.js, React Native, Browser) when you need the direct value. Imported as a default export.
2.  **CSS Variable Strings (Browser JS):** Provides strings formatted as CSS variables (`var(--h-font-...)`). Use this in browser JavaScript (React, Vue, etc.) when applying styles directly that leverage CSS variables. Imported as a named `font` export.
3.  **CSS Definitions:** Provides the actual CSS custom property definitions for family, size, and weight in a single file (`font.css`). This must be imported for the CSS Variable Strings (Browser JS) format to work correctly.

### JS/TS Usage (`/js`)

Import the default `font` object from `@utilitywarehouse/hearth-tokens/js/font` to access the raw values directly.

```js
import { font } from '@utilitywarehouse/hearth-tokens/js';

const bodyFont = font.family.body; // 'DM Sans'
const headingFont = font.family.heading; // 'Comic Hams'
const baseSize = font.size[100]; // 16
const largeSize = font.size[400]; // 30
const regularWeight = font.weight.regular; // 400
const boldWeight = font.weight.bold; // 700

// Example usage in React Native
const styles = StyleSheet.create({
  heading: {
    fontFamily: font.family.heading,
    fontSize: font.size[500], // 36
    fontWeight: font.weight.bold, // 700
  },
  bodyText: {
    fontFamily: font.family.body,
    fontSize: font.size[100], // 16
    fontWeight: font.weight.regular, // 400
  },
});

// Example usage in browser JS for non-CSS contexts
const detailFont = font.family.detail; // 'DM Mono'
```

### JS/TS CSS Variable String Usage (Browser JS - `/browser`)

Import the named `font` object from `@utilitywarehouse/hearth-tokens/browser/font` to access CSS variable strings. **Requires the combined CSS definitions (`font.css`) to be imported separately (see next section).**

```js
import { font } from '@utilitywarehouse/hearth-tokens/browser'; // Note: named import

const bodyFontVar = font.family.body; // 'var(--h-font-family-body)'
const headingFontVar = font.family.heading; // 'var(--h-font-family-heading)'
const baseSizeVar = font.size[100]; // 'var(--h-font-size-100)'
const largeSizeVar = font.size[400]; // 'var(--h-font-size-400)'
const regularWeightVar = font.weight.regular; // 'var(--h-font-weight-regular)'
const boldWeightVar = font.weight.bold; // 'var(--h-font-weight-bold)'


// Example usage in a React component's style prop
const MyComponent = () => (
  <div>
    <h1 style={{
        fontFamily: font.family.heading,
        fontSize: font.size[500],
        fontWeight: font.weight.bold,
      }}>
      Heading Text
    </h1>
    <p style={{
        fontFamily: font.family.body,
        fontSize: font.size[100],
        fontWeight: font.weight.regular,
       }}>
      Body text using CSS variables via JS.
    </p>
  </div>
);

// Example usage with styled-components
import styled from 'styled-components';

const StyledHeading = styled.h2\`
  font-family: \${font.family.heading};
  font-size: \${font.size[400]};
  font-weight: \${font.weight.semibold};
\`;

const StyledCode = styled.code\`
  font-family: \${font.family.detail};
  font-size: \${font.size[90]};
\`;
```

### CSS Usage (`/css`)

Import the combined `font.css` file in your application's entry point (e.g., main JS/TS file or global CSS file). This file defines the actual CSS custom properties for family, size, and weight (e.g., `--h-font-family-body: dm sans;`, `--h-font-size-100: 1rem;`, `--h-font-weight-regular: 400;`). **This step is necessary for the CSS Variable String (Browser JS) format to function correctly.** It also allows you to use the variables directly in your standard CSS files.

```js
// Import in your main JS/TS file or CSS entry point
import '@utilitywarehouse/hearth-tokens/css/font.css';
```

```css
/* Example usage directly in a CSS file */
body {
  font-family: var(--h-font-family-body);
  font-size: var(--h-font-size-100);
  font-weight: var(--h-font-weight-regular);
}

h1,
h2,
h3 {
  font-family: var(--h-font-family-heading);
  font-weight: var(--h-font-weight-bold);
}

h1 {
  font-size: var(--h-font-size-600);
}
h2 {
  font-size: var(--h-font-size-500);
}

code,
pre {
  font-family: var(--h-font-family-detail);
  font-size: var(--h-font-size-90);
}

/* Example using CSS Modules or similar */
:root {
  --component-heading-font: var(--h-font-family-heading);
  --component-heading-size: var(--h-font-size-400);
  --component-heading-weight: var(--h-font-weight-semibold);
}

.title {
  font-family: var(--component-heading-font);
  font-size: var(--component-heading-size);
  font-weight: var(--component-heading-weight);
}
```

## Available Font Family Tokens

| Token Key | Value                    | CSS Variable              |
| :-------- | :----------------------- | :------------------------ |
| body      | {fontJs?.family.body}    | `--h-font-family-body`    |
| detail    | {fontJs?.family.detail}  | `--h-font-family-detail`  |
| heading   | {fontJs?.family.heading} | `--h-font-family-heading` |
