# Border (Radius & Width)

Hearth Tokens provides consistent sets of border radius and border width values for styling element borders and corners. These tokens ensure uniform appearance across your projects.

## Getting Started

To use the border tokens in your project, you can import them from the `@utilitywarehouse/hearth-tokens` package. The tokens are available in multiple formats depending on your needs:

1.  **Raw Values (JS):** Provides raw pixel values for radius and width. Use this in any JavaScript environment (Node.js, React Native, Browser) when you need the direct numerical value. Imported separately for radius and width.
2.  **CSS Variable Strings (Browser JS):** Provides strings formatted as CSS variables (`var(--h-border-radius-...)`, `var(--h-border-width-...)`). Use this in browser JavaScript (React, Vue, etc.) when applying styles directly that leverage CSS variables. Imported via a combined `border` object.
3.  **CSS Definitions:** Provides the actual CSS custom property definitions for both radius and width in a single file (`border.css`). This must be imported for the CSS Variable Strings (Browser JS) format to work correctly.

### JS/TS Usage (`/js`)

Import the `borderRadius` and `borderWidth` objects separately from their respective `/js` paths (as default exports) to access the raw values directly.

```js
import { borderRadius, borderWidth } from '@utilitywarehouse/hearth-tokens/js';

const smallRadius = borderRadius['sm']; // 6
const thinBorder = borderWidth[1]; // 1

// Example usage in React Native
const styles = StyleSheet.create({
  container: {
    borderRadius: borderRadius['md'], // 8
    borderWidth: borderWidth[1], // 1
    borderColor: 'black', // Don't forget color and style!
    borderStyle: 'solid',
  },
});

// Example usage in browser JS for calculations or non-CSS contexts
const cornerSize = borderRadius['lg']; // 12
const borderSize = borderWidth[2]; // 2
```

### JS/TS CSS Variable String Usage (Browser JS - `/browser`)

Import the combined `border` object from `@utilitywarehouse/hearth-tokens/browser` to access CSS variable strings for both radius and width. **Requires the combined CSS definitions (`border.css`) to be imported separately (see next section).**

```js
import { border } from '@utilitywarehouse/hearth-tokens/browser';

const smallRadiusVar = border.radius['sm']; // 'var(--h-border-radius-sm)'
const largeRadiusVar = border.radius['lg']; // 'var(--h-border-radius-lg)'
const thinBorderVar = border.width[1]; // 'var(--h-border-width-1)'
const thickBorderVar = border.width[2]; // 'var(--h-border-width-2)'

// Example usage in a React component's style prop
const MyComponent = () => (
  <div style={{
    borderRadius: border.radius['md'], // var(--h-border-radius-md)
    borderWidth: border.width[1], // var(--h-border-width-1)
    borderStyle: 'solid',
    borderColor: 'black'
   }}>
    Content using CSS variables via JS
  </div>
);

// Example usage with styled-components
import styled from 'styled-components';

const StyledBox = styled.div\`
  border-radius: \${border.radius['lg']}; /* var(--h-border-radius-lg) */
  border-width: \${border.width[2]}; /* var(--h-border-width-2) */
  border-style: solid;
  border-color: grey;
\`;
```

### CSS Usage (`/css`)

Import the combined `border.css` file in your application's entry point (e.g., main JS/TS file or global CSS file). This file defines the actual CSS custom properties for both radius (e.g., `--h-border-radius-md: 8px;`) and width (e.g., `--h-border-width-1: 1px;`). **This step is necessary for the CSS Variable String (Browser JS) format to function correctly.** It also allows you to use the variables directly in your standard CSS files.

```js
// Import in your main JS/TS file or CSS entry point
import '@utilitywarehouse/hearth-tokens/css/border.css';
```

```css
/* Example usage directly in a CSS file */
.my-card {
  border-radius: var(--h-border-radius-lg);
  border-width: var(--h-border-width-1);
  border-style: solid;
  border-color: blue;
}

/* Example using CSS Modules or similar */
:root {
  --component-border-radius: var(--h-border-radius-sm);
  --component-border-width: var(--h-border-width-2);
}

.input-field {
  border-radius: var(--component-border-radius);
  border-width: var(--component-border-width);
  border-style: solid;
  border-color: green;
}
```

## Available Border Radius Tokens

The following table lists the available border radius tokens, their corresponding values (from the JS export), and the associated CSS variable names.

| Token Key | Value (px)             | CSS Variable             |
| :-------- | :--------------------- | :----------------------- |
| xs        | {borderRadiusJs?.xs}   | `--h-border-radius-xs`   |
| sm        | {borderRadiusJs?.sm}   | `--h-border-radius-sm`   |
| md        | {borderRadiusJs?.md}   | `--h-border-radius-md`   |
| lg        | {borderRadiusJs?.lg}   | `--h-border-radius-lg`   |
| xl        | {borderRadiusJs?.xl}   | `--h-border-radius-xl`   |
| full      | {borderRadiusJs?.full} | `--h-border-radius-full` |
| none      | {borderRadiusJs?.none} | `--h-border-radius-none` |

## Available Border Width Tokens

The following table lists the available border width tokens, their corresponding pixel values (from the JS export), and the associated CSS variable names.

| Token Key | Value (px)           | CSS Variable         |
| :-------- | :------------------- | :------------------- |
| 1         | {borderWidthJs?.[1]} | `--h-border-width-1` |
| 2         | {borderWidthJs?.[2]} | `--h-border-width-2` |
