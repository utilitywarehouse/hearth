## Color Scales

Each color scale provides a range of shades from lightest to darkest, with consistent naming across all colors.

<ColorPalette>
  {Object.keys(color)
    .filter(key => !['light', 'dark'].includes(key))
    .map(key => (
      <>
        <ColorItem
          key={`${key}-1`}
          title={key}
          subtitle={`${key}`}
          colors={Object.keys(color[key])
            .slice(0, 6)
            .reduce((acc, subKey) => {
              acc[subKey] = color[key][subKey];
              return acc;
            }, {})}
        />
        {Object.keys(color[key]).length > 6 && (
          <ColorItem
            key={`${key}-2`}
            title=""
            subtitle=""
            colors={Object.keys(color[key])
              .slice(6, 12)
              .reduce((acc, subKey) => {
                acc[subKey] = color[key][subKey];
                return acc;
              }, {})}
          />
        )}
        {Object.keys(color[key]).length > 12 && (
          <ColorItem
            key={`${key}-3`}
            title=""
            subtitle=""
            colors={Object.keys(color[key])
              .slice(12, 18)
              .reduce((acc, subKey) => {
                acc[subKey] = color[key][subKey];
                return acc;
              }, {})}
          />
        )}
      </>
    ))}
</ColorPalette>
