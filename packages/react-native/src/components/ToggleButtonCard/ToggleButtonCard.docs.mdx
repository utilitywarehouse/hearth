# ToggleButtonCard

The ToggleButtonCard component presents users with predefined choices and enables them to select only one option. It is commonly used for providing a single-choice selection in forms or surveys.

## Usage

- A `ToggleButtonCard` component is composed of a `ToggleButtonCardGroup`, `ToggleButtonCardIndicator`, `ToggleButtonCardLabel`.
- `ToggleButtonCard` is used for a single selection
- `ToggleButtonCardGroup` is the building block for `ToggleButtonCard`

<UsageWrap>
  <Center>
    <ToggleButtonCardGroup aria-label="ToggleButtonCard Group" nativeID="ToggleButtonCard-group">
      <ToggleButtonCard
        value="Label 1"
        aria-label="Label 1"
        label="Option 1"
      />
      <ToggleButtonCard
        value="Label 2"
        aria-label="Label 2"
        label="Option 2"
      />
    </ToggleButtonCardGroup>

  </Center>
</UsageWrap>

```tsx
import {
  ToggleButtonCard,
  ToggleButtonCardGroup,
  ToggleButtonCardLabel,
  ToggleButtonCardIndicator,
} from '@utilitywarehouse/native-ui';

const MyComponent = () => {
  const [value, setValue] = React.useState('Option 1');
  return (
    <ToggleButtonCardGroup
      aria-label="ToggleButtonCard Group"
      value={value}
      onChange={setValue}
      nativeID="ToggleButtonCard-group"
    >
      <ToggleButtonCard value="Option 1" aria-label="Option 1" label="Option 1" />
      <ToggleButtonCard value="Option 2" aria-label="Option 2" label="Option 2" />
    </ToggleButtonCardGroup>
  );
};
```

## Props

| Property             | Type                                | Default   | Description                                                                                          |
| -------------------- | ----------------------------------- | --------- | ---------------------------------------------------------------------------------------------------- |
| `value`              | string                              | -         | The value to be used in the radio input. This is the value that will be returned on form submission. |
| `onChange`           | function                            | -         | Function called when the state of the radio changes.                                                 |
| `disabled`           | bool                                | false     | To manually set disable to the radio.                                                                |
| `validationStatus`   | `'valid' \| 'invalid' \| 'initial'` | -         | The validation status of the radio.                                                                  |
| `disabled`           | `bool`                              | `false`   | To manually set disable to the radio.                                                                |
| `label`              | `string`                            | -         | The label to be displayed next to the radio.                                                         |
| `helperText`         | `string`                            | -         | The helper text to be displayed below the radio.                                                     |
| `helperIcon`         | `React.ComponentType`               | -         | The icon to be displayed next to the helper text.                                                    |
| `invalidText`        | `string`                            | -         | The invalid text to be displayed below the radio.                                                    |
| `validText`          | `string`                            | -         | The valid text to be displayed below the radio.                                                      |
| `showValidationIcon` | `boolean`                           | `true`    | Whether to show the validation icon.                                                                 |
| `type`               | `'default' \| 'tile'`               | `default` | The type of the radio.                                                                               |

## Components

### `ToggleButtonCard`

Contains all ToggleButtonCard related layout style props and actions. It inherits all the properties of React Native's [View component](https://reactnative.dev/docs/view).

### `ToggleButtonCardTile`

Contains all ToggleButtonCard related layout style props and actions. It inherits all the properties of React Native's [View component](https://reactnative.dev/docs/view).

### `ToggleButtonCardIndicator`

Contains all Indicator related layout style props and actions. It inherits all the properties of React Native's [View component](https://reactnative.dev/docs/view).

### `ToggleButtonCardIcon`

Contains all Icon related layout style props and actions.

### `ToggleButtonCardLabel`

Contains all Label related layout style props and actions. It inherits all the properties of React Native's [Text component](https://reactnative.dev/docs/text).

### `ToggleButtonCardGroup`

Contains all Group related layout style props and actions. It inherits all the properties of React Native's [View component](https://reactnative.dev/docs/view).

| Property             | Type                                | Default   | Description                                                                    |
| -------------------- | ----------------------------------- | --------- | ------------------------------------------------------------------------------ |
| `value`              | string                              | -         | The value of the radio group.                                                  |
| `onChange`           | function                            | -         | The callback fired when any children ToggleButtonCard is checked or unchecked. |
| `disabled`           | bool                                | -         | To manually set disable to the radio group.                                    |
| `validationStatus`   | `'valid' \| 'invalid' \| 'initial'` | -         | The validation status of the radio group.                                      |
| `label`              | `string`                            | -         | The label to be displayed above the radio group.                               |
| `helperText`         | `string`                            | -         | The helper text to be displayed below the radio group.                         |
| `helperIcon`         | `React.ComponentType`               | -         | The icon to be displayed next to the helper text.                              |
| `invalidText`        | `string`                            | -         | The invalid text to be displayed below the radio group.                        |
| `validText`          | `string`                            | -         | The valid text to be displayed below the radio group.                          |
| `showValidationIcon` | `boolean`                           | `true`    | Whether to show the validation icon.                                           |
| `type`               | `'default' \| 'tile'`               | `default` | The type of the radio group.                                                   |
| `direction`          | `'row' \| 'column'`                 | `column`  | The direction of the radio group.                                              |
| `gap`                | `string`                            | -         | The gap between the radio group items.                                         |

## Examples

### `ToggleButtonCardGroup` component

The `ToggleButtonCardGroup` component is used to group multiple `ToggleButtonCard` components together.
You can pass props like `label` and `helperText` to the `ToggleButtonCardGroup` component to provide context for the group of radios.

<UsageWrap>
  <Center>
    <ToggleButtonCardGroup
      aria-label="ToggleButtonCard Group"
      label="ToggleButtonCard Group Label"
      helperText="This is the supporting text"
      invalidText="This is an invalid text"
      validText="This is a valid text"
      validationStatus="invalid"
    >
      <ToggleButtonCard value="option-1" aria-label="Label 1" label="Option 1" />
      <ToggleButtonCard value="option-2" aria-label="Label 2" label="Option 2" />
      <ToggleButtonCard value="option-3" aria-label="Label 3" label="Option 3" />
      <ToggleButtonCard value="option-4" aria-label="Label 4" label="Option 4" />
    </ToggleButtonCardGroup>

  </Center>
</UsageWrap>

```tsx
import { ToggleButtonCard, ToggleButtonCardGroup } from '@utilitywarehouse/native-ui';

const MyComponent = () => {
  const [value, setValue] = React.useState('option-1');
  return (
    <ToggleButtonCardGroup
      aria-label="ToggleButtonCard Group"
      value={value}
      onChange={setValue}
      label="ToggleButtonCard Group Label"
      helperText="This is the supporting text"
      invalidText="This is an invalid text"
      validText="This is a valid text"
    >
      <ToggleButtonCard value="option-1" aria-label="Label 1" label="Option 1" />
      <ToggleButtonCard value="option-2" aria-label="Label 2" label="Option 2" />
      <ToggleButtonCard value="option-3" aria-label="Label 3" label="Option 3" />
      <ToggleButtonCard value="option-4" aria-label="Label 4" label="Option 4" />
    </ToggleButtonCardGroup>
  );
};
```
