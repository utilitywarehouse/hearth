# Pill Group

A container component that groups multiple `Pill` components together for filtering and categorization. It provides layout control with optional wrapping behavior.

- [Usage](#usage)
- [Examples](#examples)
- [API](#api)
- [Accessibility](#accessibility)
- [Best Practices](#best-practices)

## Usage

The `PillGroup` component is used to organize multiple `Pill` components. It manages the layout and spacing between pills, with an option to enable wrapping for responsive layouts.

### Basic Usage

```tsx
import { Pill, PillGroup } from '@hearth/react-native';

<PillGroup wrap={true}>
  <Pill label="All" selected={false} onPress={handlePress} />
  <Pill label="New" icon={StarMediumIcon} selected={true} onPress={handlePress} />
  <Pill label="Read" selected={false} onPress={handlePress} />
</PillGroup>
```

## API

### PillGroup Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | Required | `Pill` components to group together |
| `wrap` | `boolean` | `false` | Whether pills should wrap to multiple lines when they overflow |
| ...rest | `ViewProps` | - | All standard View props are supported |

## Accessibility

- All accessibility features are handled by the individual `Pill` components
