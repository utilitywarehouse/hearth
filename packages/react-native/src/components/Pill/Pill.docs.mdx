# Pill

A compact, interactive component for filtering and categorizing content. Pills can display text with optional icons and toggle between selected and unselected states.

**Note:** The `Pill` component is designed to be used within `PillGroup`. For typical usage patterns and examples, see the [PillGroup documentation](?path=/docs/components-pill-group--docs).

- [Usage](#usage)
- [Best Practices](#best-practices)
- [Examples](#examples)
- [API](#api)

## Usage

The `Pill` component is typically used within a `PillGroup` for filtering, categorization, or selection. It supports both text-only and icon + text combinations.

### Basic Usage

```tsx
import { Pill, PillGroup } from '@hearth/react-native';

<PillGroup wrap={true}>
  <Pill label="New" icon={ElectricityMediumIcon} selected={false} onPress={handlePress} />
  <Pill label="Read" selected={true} onPress={handlePress} />
</PillGroup>
```

## Best Practices

1. **Use within PillGroup** - Pills should typically be grouped using the `PillGroup` component
1. **Handle state properly** - Manage selected state in your component to control which pills are active

## API

### Pill Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | Required | The text content of the pill |
| `icon` | `ComponentType<any>` | - | Optional icon component to display before the label |
| `selected` | `boolean` | `false` | Whether the pill is in selected state |
| `onPress` | `function` | - | Callback fired when the pill is pressed |
| ...rest | `PressableProps` | - | All standard Pressable props are supported |
