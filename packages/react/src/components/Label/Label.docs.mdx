## API

This component is based on the `label` element, and supports the following common props:

- <StorybookLink to="common-props-margin">Margin</StorybookLink>
- <StorybookLink to="common-props-text">Text transform</StorybookLink>

| Prop                | Type                  |
| ------------------- | --------------------- |
| `as`                | `"label" \| "span"`   |
| `labelVariant`      | `"body" \| "heading"` |
| `disabled`          | `boolean`             |
| `disableUserSelect` | `boolean`             |
