## Sizes

The size prop is not responsive, as the sizes themselves are already responsive
where necessary.

## API

This component is based on the `h2` element.

| Prop      | Type                                        | Description                                                                                                                                    | Default |
| --------- | ------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- | ------- |
| `as`      | `"h1" \| "h2" \| "h3" \| "h4"`              | Shorthand for changing the default rendered element into a semantically appropriate alternative. Cannot be used in combination with `asChild`. | `"h2"`  |
| `asChild` | `boolean`                                   | Change the default rendered element for the one passed as a child, merging their props and behavior.                                           | `false` |
| `size`    | `'sm' \| 'md' \| 'lg' \| 'xl'`              | Set the text size styles.                                                                                                                      | `"md"`  |
| `align`   | `Responsive<"left" \| "center" \| "right">` | Set the text-align.                                                                                                                            |         |
