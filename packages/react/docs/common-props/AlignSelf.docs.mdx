The align-self CSS property overrides a grid or flex item's align-items value.

```tsx
<Flex alignItems="center">
  <Box />
  <Box alignSelf="start" />
  <Box alignSelf={{ mobile: 'end', desktop: 'center' }} />
</Flex>
```
