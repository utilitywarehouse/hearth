  # Order
  <ViewMarkdownLink to="docs/common-props/order" />

The order prop sets the order for Flex & Grid items, it is responsive and
accepts any valid string value.

```tsx

<Box order="1" />
<Flex order="-1" />
<Grid order="0" />
```
